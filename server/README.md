# ShopWise - Serveur

ShopWise est un système de recommandation et de comparaison de prix multi-plateforme conçu pour améliorer l'expérience utilisateur sur diverses plateformes e-commerce. Il analyse les produits et propose des suggestions pertinentes.

## Installation

1. Créer un environnement virtuel :
   ```bash
   python -m venv venv
   ```
2. Activer l'environnement :
   - Windows : `venv\Scripts\activate`
   - Linux/Mac : `source venv/bin/activate`
3. Installer les dépendances :
   ```bash
   pip install -r requirements.txt
   ```

## Utilisation

Pour lancer le serveur Flask :
```bash
python app.py
```
